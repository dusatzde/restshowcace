OAuth2 autentification and authorization

  • Authorization Server, Resource Server, Client (all in one app)
  • Grant_type password
  • How to get access token?
    • User Admin with password "password" (authorization ADMIN )
    • POST: https://localhost:8443/oauth/token?grant_type=password&username=Admin&password=password
  • How to use access token?
    • List all users (only for authorization ADMIN )
    • https://localhost:8443/api/v1/users/?access_token=1219088d-986c-416d-8de7-440901b88a6a
